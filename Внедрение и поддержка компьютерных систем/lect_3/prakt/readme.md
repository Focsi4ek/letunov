# Тестирование кружки для горячих напитков

## Black Box Testing

Проверка функциональности без учёта внутренней структуры изделия.

* Сохраняет температуру напитка (горячий остаётся горячим, холодный — холодным)?
* Не протекает при наполнении?
* Подходит для разных напитков (кофе, чай, какао, вода)?
* Удобно держать в руке?
* Можно ли пить, не проливая?

## Automated Testing

Автоматизированные проверки для массового производства и эксплуатации.

* Выдержит ли многократное использование в кофейном автомате?
* Сохранит ли целостность при автоматической мойке в посудомоечной машине (несколько циклов)?
* Устойчивость к перепадам температур при автоматическом наполнении (от холодного к горячему)?
* Совместимость с линиями автоматической упаковки и маркировки?

## Functional Testing

Проверка основных функций и характеристик кружки.

* Основная функция: удерживает жидкость без протечек?
* Ручка надёжно прикреплена и не отваливается при использовании?
* Края кружки гладкие, не царапают губы?
* Объём соответствует заявленному ($250\ \text{мл}$, $350\ \text{мл}$ и т. д.)?
* Напиток не проливается при наклоне (до определённого угла)?
* Легко наполняется из чайника/кофеварки?

## Validation

Проверка соответствия реальным условиям использования и безопасности.

* Не деформируется при контакте с кипятком?
* Не выделяет вредных веществ при нагреве?
* Материал устойчив к царапинам от ложки?
* Выдерживает падение с высоты $1\ \text{м}$ на твёрдую поверхность?
* Сохраняет цвет и рисунок после многократного мытья?
* Влагостойкость: не впитывает запахи и красители от напитков?
* Термостойкость: выдерживает диапазон температур от $-20^\circ\text{C}$ до $+100^\circ\text{C}$?
* Соответствует ли форма эргономическим требованиям (удобно держать)?
* Инструкция по уходу и использованию понятна и доступна?

## Verification

Проверка на соответствие техническим спецификациям и документации.

* Соответствует ли материал (керамика, стекло, нержавеющая сталь, пластик) заявленным характеристикам?
* Цвет и дизайн соответствуют макету?
* Маркировка содержит всю необходимую информацию (объём, материал, рекомендации по уходу)?
* Упаковка защищает кружку при транспортировке?
* Вес соответствует спецификации?

## Usability Testing

Оценка удобства и комфорта использования.

* Удобно ли держать кружку одной рукой?
* Достаточно ли широкая горловина для комфортного питья и мытья?
* Легко ли мыть вручную и в посудомоечной машине?
* Удобно ли хранить (занимает мало места, можно ставить одну в другую)?
* Не скользит в руке (антискользящее покрытие на дне)?
* Можно ли использовать в микроволновой печи (если заявлено)?

## Compatibility Testing

Проверка совместимости с различными условиями и средами.

* Совместима ли с разными типами напитков (кислые, сладкие, с молоком)?
* Не меняет вкус напитка?
* Безопасна для детей (отсутствие острых краёв, нетоксичный материал)?
* Подходит ли для использования в микроволновке и морозильной камере (если заявлено)?
* Совместима с аксессуарами (подставки, крышки)?
* Срок службы соответствует заявленному?

## Performance Testing

Нагрузочное тестирование и проверка долговечности.

* Сколько циклов нагрева/охлаждения выдержит до появления трещин?
* Сколько раз можно уронить с высоты $1\ \text{м}$ без повреждений?
* Как долго сохраняет температуру напитка (в минутах)?
* Износостойкость поверхности: сколько циклов мытья в посудомоечной машине выдержит до появления царапин?
* Устойчивость к истиранию рисунка/надписи?
* Предел прочности на сжатие (можно ли надавить без деформации)?
* Проверка термошока: выдержит ли резкий перепад температур (налить кипяток в холодную кружку)?
